## 如何参与翻译

1. fork本项目
2. 在本地安装 gitbook, `npm install gitbook-cli -g`
2. 修改_src目录下的 markdown 文件
3. 执行`gitbook build`编译book
4. push 到 github
5. 提交 PullRequest

参与的同学在[issues](https://github.com/jirengu-inc/javascript.info/issues/1)里回复认领文章明细
